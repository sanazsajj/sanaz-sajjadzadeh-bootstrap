Element

Ett element är en del av ett HTMl dokument som representerar olika delar i dokumentet, ex. en title element representerar titeln. De flesta HTML element har både start tagg och slut tagg och dem kan också ha ett eller flera attribut som bestämmer andra egenskaper elementet har.

Så här kan ett element se ut:

HTML

HTML står för HyperText Markup Language och är ett märkspråk för att beskriva text och innehåll på en webbsida. Man anger vilka delar som är rubriker, vanliga textstycken, listor och länkar och sedan tolkar webbläsaren det och visar de på olika sätt. Med CSS kan man sen själv bestämma hur de ska visas.

Tillsammans med CSS och Javascript utgör de, de tre viktigaste teknikerna.

ID

Ett ID används för att specificera ett enda HTML element i en CSS-regel. Ett ID ska vara unikt och får bara användas en gång i varje HTML-dokument. Attributet används för att ange en specifik stildeklaration till bara ett element.

Meta

Meta taggen definierar metadatan om ett HTML-dokument, vilket är informationen om data. Taggen skrivs alltid inuti head elementet och används oftast för att specificera teckenuppsättning, beskrivning av sidan, nyckelord, författaren av dokumentet och visningsinställningar.

Metadatan används av webbläsare, sökmotorer och andra webbservice men syns inte på sidan.

Responsive

En responsiv sida är en sida som är designad så att layouten automatisk förändras beroende på vilken skärmstorleken eller skärmupplösningen som besökaren har.

Innehållet på hemsidan, så som bilder och text, skalas då för att passa oavsett om man är på telefonen eller datorn.

http

Http är en förkortning för Hypertext Transfer Protocol och är ett kommunikationsprotokoll för att begära en resurs från en webbserver. Det används i bland annat webbläsare för att hämta data från internet ex. texter eller bilder.

https

Https är en förkortning för Hypertext Transfer Protocol Secure och innebär att all kommunikation mellan webbläsaren och hemsidan är krypterad.

Compressed data

Compressed data är data som omkodas så att färre informationsbärande enheter behöver används och detta görs för att ex. spara lagringsutrymme på disk.

CDN

CDN är en förkortning för Content Delivery Network och är en högt distribuerad plattform av servrar som hjälper till att minska att innehållet på webbsidan laddas upp långsamt genom att minska det fysiska avståndet mellan servern och användaren.

Local bootstrap

Bootstrap är en samling av återanvändbara koder skrivna i HTML, CSS och Javascript vilket gör så att det går snabbare att utveckla en hemsida då man inte måste skriva all CSS kod själv.

JQuery

Jquery är ett Javascript-bibliotek som förenklar användningen av Javascript genom att ta vanliga uppgifter som annars kräver flera rader av kod och förkortar dem så att de endast kräver en enda rad av kod.

Flexbox

Flexbox är en webb layout modell som tillåter responsiva element inuti en container att automatisk ändras beroende på skärmstorleken. Den förenklar också många andra saker som att göra så att kolumner har samma höjd och centrerar element.

Grid

Grid är en layout modell som har ett rutnät baserat layout system, med rader och kolumner, vilket gör det lättare att designa hemsidor.

Code snippet

Code snippets beskriver en liten del av återanvändbar källkod, maskinkod eller text. De gör så att man slipper skriva samma kod flera gånger när man programmerar.
